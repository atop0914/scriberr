# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-03-11

### Added
- Initial release
- Whisper model integration (tiny, base, small, medium, large, large-v2, large-v3)
- Multiple audio format support (MP3, WAV, M4A, OGG, FLAC, AAC, WMA, OPUS)
- Multiple output formats (text, JSON, SRT, VTT)
- Batch processing with glob patterns
- Recursive directory scanning
- Config file support (YAML)
- Comprehensive CLI with urfave/cli
- Model download and caching
- Timestamp support for subtitles

### Features
- Language auto-detection
- Translation to English
- Multi-threaded transcription
- Verbose and quiet output modes

## [0.0.1] - 2024-03-06

### Added
- Initial project setup
- Basic project structure
